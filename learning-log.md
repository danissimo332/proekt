## 23.03.2026

Базы данных — это системы для хранения и управления данными.

Реляционные базы данных (SQL):
- данные хранятся в таблицах
- есть связи между таблицами
- используется язык SQL
- примеры: PostgreSQL, MySQL, MsSQL

NoSQL базы данных:
- нет строгой структуры
- данные часто хранятся в формате JSON
- более гибкие
- примеры: MongoDB, Redis, ClickHouse

PostgreSQL — это реляционная база данных.
- хранит данные в таблицах
- поддерживает сложные SQL-запросы
- надежная и подходит для больших проектов
- поддерживает связи между данными

Инструменты для работы с БД — это программы, через которые можно работать с базой данных.

Примеры:
- DBeaver
- pgAdmin
- MySQL Workbench

Что они делают:
- позволяют писать SQL-запросы
- показывают таблицы и данные
- упрощают работу с БД

Умное ЖКХ — это использование технологий для управления домами и коммунальными услугами.

Примеры:
- умные счетчики воды и электричества
- автоматический сбор данных
- системы мониторинга

Что дает:
- автоматизацию процессов
- точные данные
- уменьшение ручной работы
